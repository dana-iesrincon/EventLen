# EventLen

-Título del Proyecto: # EventLen (Project Code: EVENTS).

-Descripción: Plataforma para centralizar la gestión de eventos culturales y deportivos en Gran Canaria.

-Propósito del MVP: Resolver la fragmentación de publicaciones y automatizar inscripciones según la capacidad real.

-Stack Tecnológico: Python 3.x, Django 4.x, PostgreSQL y Bootstrap 5.

-Roles: User, Organizer y Admin.
